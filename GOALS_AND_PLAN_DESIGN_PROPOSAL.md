# 基本方針・長期目標と計画ページの設計案

## 現状の課題

- 「基本方針・長期目標」に何を書けばいいか分からない
- 計画ページ（`/instagram/plan`）との違いが分かりにくい
- 計画ページを残すべきか削除すべきか判断に迷う

---

## 案1: 計画ページを残す（推奨）

### 基本方針・長期目標の例

**「数ヶ月〜数年変わらない、ビジネスの根幹となる方針」**

#### 良い例 ✅

```
ブランド認知度の向上
顧客との長期的な関係構築
業界での専門性・信頼性の確立
地域No.1の認知度獲得
```

#### 悪い例 ❌（月ごとに変わるもの）

```
今月は新商品の認知度向上に重点を置く
3月のキャンペーンで売上30%UP
今月のフォロワー数目標: 1,000人
```

### 計画ページ（`/instagram/plan`）の役割

**「月ごとに変わる具体的な目標と戦略」**

#### 例

```
今月の目標:
- 新商品「○○」の認知度向上（リーチ10,000人）
- エンゲージメント率を5%に向上
- ストーリーズの活用で顧客との距離を縮める

今月の課題:
- 投稿頻度が低い（週2回→週4回に増やす）
- ハッシュタグ戦略が見直しが必要
```

### メリット

- ✅ 基本方針は一度設定すれば長期間使える
- ✅ 月ごとに柔軟に目標を変更できる
- ✅ AI生成時に優先順位が明確（月次目標 > 基本方針）
- ✅ ユーザーが「今月何をすべきか」を明確に意識できる

### デメリット

- ⚠️ 2箇所で目標を設定する必要がある
- ⚠️ ユーザーが混乱する可能性

---

## 案2: 計画ページを削除（シンプル化）

### 基本方針・長期目標のみにする

**「ビジネスの根幹となる方針（変更頻度: 低い）」**

#### 例

```
ブランド認知度の向上
顧客との長期的な関係構築
業界での専門性・信頼性の確立
```

### メリット

- ✅ シンプルで分かりやすい
- ✅ 設定箇所が1つだけ
- ✅ ユーザーの混乱が少ない

### デメリット

- ⚠️ 月ごとの具体的な目標を設定できない
- ⚠️ AI生成時に「今月の重点」が反映されにくい
- ⚠️ PDCAサイクルを回しにくい

---

## 案3: 基本方針を「ビジネスミッション」に変更

### 名称変更

**「基本方針・長期目標」→「ビジネスミッション・価値観」**

### 例

```
私たちのミッション:
- 毎日のコーヒーを特別な体験に
- お客様との長期的な関係を大切にする
- 地域コミュニティの活性化に貢献する

SNSで実現したいこと:
- ブランドの世界観を伝える
- お客様との距離を縮める
- 新商品・サービスの認知拡大
```

### メリット

- ✅ 「ミッション」という言葉で長期性が明確
- ✅ 計画ページとの違いが分かりやすい

---

## 案4: 基本方針を「SNS活用の目的」に変更

### 名称変更

**「基本方針・長期目標」→「SNS活用の目的（Why SNS?）」**

### 例

```
SNSを活用する目的:
- ブランド認知度の向上
- 新規顧客の獲得
- 既存顧客との関係強化
- 業界での専門性アピール
```

### メリット

- ✅ 「Why SNS?」で目的が明確
- ✅ 計画ページの「今月の目標」と区別しやすい

---

## 推奨案: 案1（計画ページを残す）+ 案4（名称変更）

### 最終的な設計

**1. 基本方針・長期目標 → 「SNS活用の目的（Why SNS?）」に変更**

```
SNSを活用する目的:
- ブランド認知度の向上
- 新規顧客の獲得
- 既存顧客との関係強化
```

**2. 計画ページ（`/instagram/plan`）は残す**

```
今月の目標:
- 新商品「○○」の認知度向上
- エンゲージメント率5%達成

今月の課題:
- 投稿頻度を週4回に増やす
```

### 説明文の改善案

```
この目的は、あなたのビジネスがSNSで何を実現したいかを表す基本方針です。
数ヶ月〜数年変わらない、ビジネスの根幹となる目的を設定してください。

月ごとに変わる具体的な目標は、運用計画ページ（/instagram/plan）で設定できます。
```

## 実装の優先順位

### すぐに実装すべき（高優先度）

1. ✅ **説明文の改善**
   - 「SNS活用の目的」という言葉を追加
   - 計画ページとの違いを明確化

2. ✅ **プレースホルダーの改善**
   - より具体的な例を提示

### 検討が必要（中優先度）

3. ⚠️ **セクション名の変更**
   - 「基本方針・長期目標」→「SNS活用の目的（Why SNS?）」
   - ツール側と統一するか検討

4. ⚠️ **計画ページの有無**
   - 残す場合: 説明文で役割を明確化
   - 削除する場合: 基本方針のみで運用

## 質問への回答

### Q: 「基本方針・長期目標」に何を書けばいい？

**A: 「数ヶ月〜数年変わらない、SNSで実現したい根本的な目的」**

例:
- ✅ ブランド認知度の向上
- ✅ 顧客との長期的な関係構築
- ✅ 業界での専門性・信頼性の確立
- ❌ 今月のフォロワー数目標（これは計画ページで設定）

### Q: 計画ページは残すべき？

**A: 残すことを推奨**

理由:
- 月ごとのPDCAサイクルを回しやすい
- AI生成時に「今月の重点」を反映できる
- ユーザーが「今月何をすべきか」を明確に意識できる

ただし、説明文で役割を明確化する必要がある。
